### 兼容系统 
- Mac OS
- Linux

### 描述 
从[某网站](https://bing.ioliu.cn/)网站下载Bing壁纸。

### 依赖 
    # scrapy 和 python3 
	$ sudo pip3 install scrapy

### 图片保存地址 
   到wallpaper/settings.py中修改第12行的路径为自己电脑上的路径以存储图片。

### 使用 
	退回到项目主目录wallpaperspider/下
	$ scrapy crawl wallpaper
